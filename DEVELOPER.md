# Developers Guide. Ish.

## First:
1. Install node.js
2. Install n (sudo npm install n -g)
3. Install yarn (sudo npm install yarn -g)

## Getting started
```
$ n 8.11.1
$ git clone (your forked repository)
$ cd companion
$ yarn
$ git submodule init
$ git submodule update
$ cd bitfocus-skeleton
$ yarn
$ cd ..
$ npm run start
```
